# GraphQL Server

## Overview
This server handles real-time features including chat and attendance using GraphQL subscriptions and mutations.

## Features
- **Real-time Chat**: Group and private messaging.
- **Attendance**: Tracking student attendance.
- **File Uploads**: Handling attachments.

## Scripts
- `npm run dev`: Starts the server (port 4001).
- `npm run chat`: Starts the chat server.

## Dependencies
- Apollo Server
- Express
- Socket.io
- Firebase Admin
